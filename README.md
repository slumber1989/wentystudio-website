# Wenty Studio 官网

一个采用 Anthropic 风格设计的展示型网站，用于介绍 AI 语音转文字工具和学习资源。

## 特点

- 🎨 温暖明亮的 Anthropic 风格设计
- 📱 完全响应式，支持移动端
- 🚀 纯静态网站，可直接部署到 GitHub Pages
- ✨ 简洁优雅的用户界面

## 本地预览

直接用浏览器打开 `index.html` 文件即可预览。

## 部署到 GitHub Pages

1. 将代码推送到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择部署分支（通常是 `main` 或 `gh-pages`）
4. 几分钟后即可通过 `https://slumber1989.github.io/wentystudio-website` 访问

## 绑定自定义域名

在仓库根目录创建 `CNAME` 文件，内容为你的域名：
```
wentystudio.com
```

然后在 Cloudflare DNS 设置中添加 CNAME 记录指向 `slumber1989.github.io`。

## 技术栈

- CSS3
- 纯原生 JavaScript（无框架依赖）
